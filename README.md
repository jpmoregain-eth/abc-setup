# ABC Setup Wizard

Agent Bear Corps - Universal Agent Setup Wizard

## Features

- **Step 1:** Agent identity (name, owner, language)
- **Step 2:** Persona selection + custom prompts
- **Step 3:** AI model provider selection (7+ providers)
- **Step 4:** Capabilities & tools selection (30+)
- **Step 5:** Confirmation & review
- **Step 6:** Generate config & launch agent

## Quick Start

```bash
pip install -r requirements.txt
python app.py
# Open http://localhost:5000
```

## Supported AI Providers

- Anthropic (Claude)
- OpenAI (GPT)
- Google (Gemini)
- Mistral AI
- Groq
- Together AI
- Custom/OpenAI-compatible

30+ tools including:
